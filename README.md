# Signup & Login System

[Live Demo](https://gungun210.github.io/login_and_signup_page/)

## About
A simple Signup & Login system built to handle user authentication with username and password.  
It demonstrates basic authentication workflow, form validation, and secure data handling.

## Features
- User Signup with username and password  
- User Login with validation  
- Password security (optional: hash / basic validation)  
- Error messages for invalid login  
- Responsive design (if web version)  
- Local storage / CSV / database support depending on project

## Tech Used

- CSV module (for storing credentials)  
- HTML
- CSS
- JavaScript (if web version)  


## How to Run
1. Clone or download the repository  
2. Ensure Python 3.x is installed  
3. Run the application
